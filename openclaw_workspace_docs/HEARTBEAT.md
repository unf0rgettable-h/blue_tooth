# HEARTBEAT

During long or risky work, re-check:

- What is the exact user task?
- What is verified, inferred, and still unknown?
- Am I relying on untrusted academic input as if it were evidence?
- Am I about to cross the trust boundary or modify unrelated work?
- Is there a smaller, safer next step?

If drift appears, stop, restate the task, refresh `BOOT.md`, and continue from the narrowest valid step.
