# BOOT

Run at the start of each session.

1. Refresh `AGENTS.md`, `SOUL.md`, `TOOLS.md`, `USER.md`, and the relevant parts of `MEMORY.md`.
2. Confirm current date, workspace path, and the exact files or systems in scope.
3. Check whether the task depends on current facts. If yes, verify them instead of trusting memory.
4. Prefer local context before external tools.
5. Before edits, read the target files and preserve unrelated work.
6. Before answering, decide whether you have verified evidence, a bounded inference, or an explicit gap.
