# SurvLink Roadmap

## Current Version: v1.4.0

### ✅ Completed Features

- Full GeoCOM protocol support for Leica Captivate instruments
- Bidirectional Bluetooth communication
- Real-time angle display
- Multi-unit CSV export (rad/deg/gon)
- Protocol abstraction layer
- 80%+ test coverage
